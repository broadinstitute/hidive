Hidive is a targeted genome co-assembler for biobank-scale long-read and short-read data. It is designed to assemble high-diversity loci from long-read data and to co-assemble these loci with short-read data. Hidive is built on top of the Skydive library, which provides a series-parallel graph representation of long-read data and a linked de Bruijn graph representation of short-read data.

Â§Prerequisites

Hidive is designed to access local files or data in Google Cloud Storage (GCS). Within certain cloud-computing environments (i.e. Terra, All of Us Researcher Workbench), access to GCS is already configured. For accessing files in GCS on your local machine, you will also need to install the Google Cloud CLI. Then, configure your Application Default Credentials (ADC). If accessing requester pays buckets set the following environment variable before running hidive commands:

 export GCS_REQUESTER_PAYS_PROJECT=<Google Project ID>
